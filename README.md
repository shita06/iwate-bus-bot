# iwate-bus-bot
# 盛岡バス時刻案内ボット（高松の池口版）

盛岡市高松の「高松の池口」バス停から盛岡駅方面へ向かうバスの時刻を、現在時刻に合わせて案内するPythonプログラムです。

## このプロジェクトの目的
岩手県交通の複雑な系統（307支線A/B、循環、あすみ野経由など）を一つのリストに統合し、「結局、次は何時にバスが来るのか」を即座に判断できるようにすることを目指しています。

## 主な機能
- **自動ダイヤ判定**: 実行時の曜日から「平日」と「休日」を自動で切り替えます。
- **複数路線の統合表示**: 系統が違っても、出発時刻が早い順に並べて表示します。
- **スマート検索**: 現在時刻を過ぎたバスは表示せず、次に来る5本を表示します。

## 使用技術
- **言語**: Python 3.12.10
- **ライブラリ**: 標準ライブラリ（datetime）のみで動作
- **管理**: GitHubによるバージョン管理

## 使い方
1. Pythonがインストールされた環境で `main.py` を実行します。
2. 現在時刻に基づいた最新のバス案内が表示されます。
